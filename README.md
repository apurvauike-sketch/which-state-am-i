# Which State Am I?

## Challenge
Mini Challenge 2 — PromptWars Virtual (Hack2Skill x Google for Developers)

## Approach
I generated a single AI image using a descriptive prompt built entirely from
visual cultural markers — local food, traditional art style, regional
architecture, and clothing — without ever naming the state directly. The goal
is for a viewer to recognize the state purely from these visual cues.

## How it works
- `index.html` displays the generated image in a simple, distraction-free card layout.
- `state.png` is the AI-generated image itself.
- No text on the page or in the image reveals the state's name, per the challenge rules.

## Assumptions
- "Building a solution" for this mini-challenge is interpreted as producing and
  hosting the creative artifact (the image + its presentation page), since this
  challenge is a prompting/creative task rather than an application-build task.
- The repository is kept minimal (two files) to stay well under the 10MB limit
  and within a single branch, as required.
